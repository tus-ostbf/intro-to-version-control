# Lesson 1: Introduction to Version Control

## Quick Summary

This lesson introduces the core concepts of version control, Git, and GitHub. Students will learn why version control is important, how to install Git, and create their first commit.

## Learning Goals

- Understand the purpose of version control.
- Install Git on local machine.
- Create a local Git repository and make a commit.

## Prerequisites

- Basic computer literacy (file navigation, text editing).
- A GitHub account (free tier).
- Git installed (installation steps covered in the lesson).
